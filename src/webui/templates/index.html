Live Input Tester

Select a controller to monitor its button presses and axis values in real time.

Serial Port Test

Try to open the JVS serial port at 115200 8N1 and report the result. Useful when the device is not responding and you want to confirm the path is correct and accessible.

JVS Bus

Two tools for inspecting the JVS bus. Probe Bus sends a RESET + ASSIGN_ADDR broadcast and listens for 2 s — any response bytes confirm a connected, powered-on board. Monitor Bus listens passively for 5 seconds without sending anything, showing whatever packets the arcade board is already transmitting. Both tools auto-detect a running service and report bus status without touching the port. When the service is stopped, the sense line is floated before listening.

GPIO Sense Line Test

Read the current logic level of the configured sense-line GPIO pin via the Linux GPIO character device. Useful when first wiring the sense line — confirms the pin is reachable and reports HIGH or LOW. Note: if the ModernJVS service is currently running it holds the GPIO line exclusively; the read will return IN USE rather than a logic level. Stop the service first to read or manually drive the pin.

Use Set HIGH / Set LOW to manually drive the pin to a known state for the chosen duration (e.g. to verify wiring with a multimeter), then the line is released automatically.

USB Device Inspector

All connected USB devices read from /sys/bus/usb/devices/. Known RS-485/serial adapters are highlighted. Shows the currently bound kernel driver — useful when a /dev/ttyUSB* node is not appearing.
